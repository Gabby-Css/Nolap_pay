# NOLAP Pay USSD Simulator (*1145#)

## Name: Benarkuu Gabriel Gbiel Junior.


## Overview
This is the final project for the Code The Vacation With Mortoti Bootcamp.
It simulates the NOLAP Pay mobile money USSD service (*1145#) entirely in the command line using Python.

Users can:
- Create accounts and log in securely
- Transfer money with dynamic fees
- Buy airtime
- Withdraw and deposit funds
- Check balances (with PIN)
- Change PINs
- Exit the app safely

All user data is stored in memory(not in a file or database).
